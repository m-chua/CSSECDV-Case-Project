# Install the packages & seed the database

`cd frontend`
`npm install`
`cd ../backend`
`npm install`
`node seeder/seedDatabase.js`

# Run
Open two terminals

Terminal #1: Must be on the frontend folder
`npm run dev`

Terminal #2: Must be on the backend folder
`node server.js`
